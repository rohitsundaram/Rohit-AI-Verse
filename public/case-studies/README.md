## PDF Format Guidelines:

- One-page format preferred
- Include company logo (if permitted)
- Highlight key metrics prominently
- Include contact information for inquiries
- Professional design matching website branding

## Creating PDFs:

You can create these PDFs using:
- Canva (recommended for design)
- Adobe InDesign
- Google Docs (export as PDF)
- Any PDF creation tool

The PDFs should be accessible at:
- `/case-studies/resume-screener-rag.pdf`
- `/case-studies/workflow-automation.pdf`
